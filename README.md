# 区块链和智能合约项目

## 概述
本项目展示了在 Sepolia 测试网络上使用 Go 和 Solidity 进行区块链交互和智能合约部署。项目包括两个主要任务：区块链读写操作和智能合约代码生成。

## 任务 1：区块链读写
### 目标
在 Sepolia 测试网络上实现基本的区块链交互，包括查询区块和发送交易。

#### 环境搭建
- 安装 Go 语言环境。
- 安装 `go-ethereum` 库。
- 注册 Infura 账户并获取 Sepolia 测试网络的 API Key。

#### 查询区块
- 使用 Go 中的 `ethclient` 连接到 Sepolia 测试网络。
- 查询指定区块号的区块信息（包括区块哈希、时间戳、交易数量等）。
- 将结果输出到控制台。

#### 发送交易
- 准备一个 Sepolia 测试网络的以太坊账户并获取其私钥。
- 使用 `ethclient` 连接到 Sepolia 测试网络。
- 构造一笔简单的以太币转账交易，指定发送方、接收方和转账金额。
- 对交易进行签名并发送到网络。
- 输出交易的哈希值。

## 任务 2：合约代码生成
### 目标
使用 `abigen` 工具自动生成 Go 绑定代码，用于与 Sepolia 测试网络上的智能合约交互。

#### 编写智能合约
- 使用 Solidity 编写一个简单的智能合约（例如计数器合约 `Counter.sol`）。
- 编译智能合约，生成 ABI 和字节码文件（例如 `Counter.abi` 和 `Counter.bin`）。

#### 使用 abigen 生成 Go 绑定代码
- 安装 `abigen` 工具。
- 根据 ABI 和字节码文件使用 `abigen` 工具生成 Go 绑定代码。

#### 使用生成的 Go 绑定代码与合约交互
- 编写 Go 代码，使用生成的绑定代码连接到 Sepolia 测试网络上的智能合约。
- 调用合约方法（例如增加计数器值）。
- 输出调用结果。
